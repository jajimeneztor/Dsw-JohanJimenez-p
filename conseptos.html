¿Que es HTML?

HTML es un lenguaje de programacion que usa una serie de codigos catalogados como etiquetas que definen los elementos que componen una pagina Web.


Etiquetas de contenido en HTML


ETIQUETASDESCRIPCION
SectionEl elemento de HTML section (<"section">) representa una sección genérica de un documento, de igual manera brinda un posicionamiento dentro del esquema de la pagina web
AsideEl elemento HTML <"aside"> representa una sección de una página que consiste en contenido que está indirectamente relacionado con el contenido principal del documento.
FooterEl Elemento HTML Footer (<"footer">) representa un pie de página para el contenido de sección más cercano o el elemento raíz de sección
ArticleEl Elemento article de HTML (<"article">) representa una composición auto-contenida en un documento, página, una aplicación o en el sitio, que se destina a distribuir de forma independiente o reutilizable.

Que es un mockup

Los Mock Ups son fotomontajes que permiten a los diseñadores gráficos y web mostrar al cliente cómo quedaran sus diseños. Además, permiten ahorrar en gastos de impresión y de montajes ya que, gracias a estas maquetas, se puede enseñar a los clientes una idea más aproximada de cómo se vería su diseño en diferentes formatos: papel, páginas web, papelería, rótulos, vinilos, carteles con resultados muy realistas y bastante aproximados a la realidad.
